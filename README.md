# Precalculus1 with Experiments (2026 Spring)
**Open Source Experiment Code for 'Precalculus1 with Experiments' at POSTECH**

This repository contains the open-source experiment code and resources for the **Precalculus1 with Experiments** course (Spring 2026) at **POSTECH** for freshman.

### 📚 Course Information
* **Korean:** [Course Announcement](https://freshman.postech.ac.kr/freshman/mus-bbs/bbs_notice.do?mode=view&articleNo=44367)
* **English:** [Course Website](https://sites.google.com/view/precalculus1-postech)

### 🚀 Quick Start
You can run the experiment code with "blanks" directly in your browser using Google Colab:
* [**Open in Colab**] [Colab](https://colab.research.google.com/drive/1nPmjYl6NbxwVk8F6J7lMs-xufir_Lt64?usp=sharing)
* Complete codes (with blanks filled) are found in github (here).

### Installation
Clone this repository into your environment.
```
git clone https://github.com/hanjun-park/precalculus1-2026
cd precalculus1-2026
```

The following basic python packages are required. (This will be updated as the course progresses.)

- python version 3.11.X, 3.12.X (recommended)

- math, matplotlib, numpy
  ```
  pip install math matplotlib numpy 
  ```
